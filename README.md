# Cv
LaTeX source code for my professional CV, designed and compiled using Overleaf.
This repository contains the LaTeX source code for my professional CV, created and compiled using Overleaf.
The document features a clean, modern layout with custom styling, icons, and structured sections to present my academic background, projects, certifications, and volunteering experience.
Features are Modern and elegant CV layout, Compact margins and custom section styling, Olive-green themed headings, FontAwesome5 icons for a polished look, Hyperlinked contact information, Profile image embedded with a styled frame, Well-organized sections for education, projects, activities, and more
This CV is designed for academic, internship, and tech-industry applications, highlighting my skills, education, and experience as a Computer Science undergraduate.
